# iStroe
An eCommerce app based on Laravel.

Author: InfiniteN [Email](mailto:liubingyao@gmail.com)

# Depdencies
* npm
* Composer
* Grunt
  * grunt-contrib-concat
  * grunt-contrib-sass
  * grunt-contrib-uglify
  * grunt-contrib-watch
  * grunt-phpunit
  * load-grunt-tasks
* Bower
  * AngularJs
  * Bootstrap-sass
  * Modernizr
  * jQuery

  * articles
    * show all articles

  * cart
    * show cart
  * checkout
